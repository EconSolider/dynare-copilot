> 实质推导，`derivation-style.md` 给出把它们排版成八节的规范。
>
> **用法**：判型后（§2 任务路由）按本文件逐 block 搭模型——家庭→厂商→政府央行→市场出清，
> **每想好一个 block 就把它的最优化问题与 FOC 落进 derivation.md 的对应小节**，再搭下一个。

> - **R2 时序 = 决定期 / 期末存量**：$t$ 期决定的存量记 $X_t$，下期才投入使用。资本 $K_t$ 是
>   *期末*存量，故 $t$ 期生产用上期决定的 $K_{t-1}$；运动律左边是期末量 $K_t$。
> - **R8 非线性优先**：默认写**原始非线性** FOC（让 Dynare 做泰勒展开），不手推线性化。

> - **不优化的主体不进 FOC**：政府支出外生、央行机械规则、纯预算恒等式（如 $T_t=G_t$）**没有
>   最优化问题**，它们的关系按性质归入「市场出清/恒等式」或「外生过程」，不编 FOC（见 §C 说明）。

## A. 家庭：动态最优化问题与一阶条件

### A.1 标准设定（CRRA 可分效用 + 债券 + 资本，封闭经济代表性家庭）

代表性家庭选择消费 $C_t$、劳动 $N_t$、债券 $B_t$、投资 $I_t$ 与资本 $K_t$，最大化期望贴现效用：

$$\max_{\{C_t,\,N_t,\,B_t,\,I_t,\,K_t\}}\;
E_0\sum_{t=0}^{\infty}\beta^t\left[\frac{C_t^{1-\sigma}-1}{1-\sigma}\;-\;\psi\frac{N_t^{1+\varphi}}{1+\varphi}\right]$$

约束一，**预算约束**（实际项，$R_t$ 为债券到期总收益，$R_t^k$ 为资本租金率，$\Pi_t$ 为厂商利润分红，$T_t$ 为一次总付税）：

$$C_t + I_t + \frac{B_t}{R_t} \;=\; W_t N_t + R_t^k K_{t-1} + B_{t-1} + \Pi_t - T_t$$

约束二，**资本积累**（左边为期末存量 $K_t$，R2）：

$$K_t = (1-\delta)K_{t-1} + I_t$$

> **时序要点**：本期出租的是上期决定的 $K_{t-1}$，故租金收入写 $R_t^k K_{t-1}$；本期决定的 $K_t$
> 下期才产出。债券 $B_t$ 同理为期末持有，本期付价 $1/R_t$、下期收 $B_t$（这里把到期值并入 $B_{t-1}$）。

### A.2 一阶条件（拉格朗日乘子 $\lambda_t$ 配预算约束）

对 $C_t,N_t,B_t,K_t$ 求导（投资 $I_t$ 通过资本运动律并入对 $K_t$ 的 FOC），消去乘子 $\lambda_t=C_t^{-\sigma}$：

- **(家庭-1) 边际效用定义**：$\;\lambda_t = C_t^{-\sigma}\;$（消费的边际效用 = 预算乘子；常作为辅助式或直接代入）。

- **(家庭-2) 劳动供给 FOC**（消费—闲暇边际替代率 = 实际工资）：
$$\psi\,N_t^{\varphi}\,C_t^{\sigma} = W_t$$

- **(家庭-3) 债券欧拉方程 Euler equation**（跨期消费选择）：
$$C_t^{-\sigma} = \beta\,R_t\,E_t\big[C_{t+1}^{-\sigma}\big]$$

- **(家庭-4) 资本欧拉方程**（投资的跨期回报；由 $\partial K_t$ 与 $\partial I_t$ 合并，无调整成本时 Tobin's $q_t=\lambda_t$）：
$$C_t^{-\sigma} = \beta\,E_t\Big[C_{t+1}^{-\sigma}\big(R_{t+1}^k + 1-\delta\big)\Big]$$

> (家庭-3)(家庭-4) 一起隐含无套利：确定性等价下 $R_t = E_t[R_{t+1}^k+1-\delta]$。
> 两条都保留时不必再写无套利式，否则方程冗余（R4）。

每条 FOC 在 derivation.md 里编 (F1)(F2)…，并在 .mod 用 `[name=]` 回指。

### A.3 常见变体（**会改方程结构 → 属闸门1 的建模分叉，先问用户**）

| 变体 | 怎么改 | 影响 |
|------|--------|------|
| **无资本**（禀赋经济 / 基础 NK） | 删 $I_t,K_t,R_t^k$ 与 (家庭-4)，预算去掉投资与租金项 | 只剩 (家庭-2)(家庭-3)；模型大幅简化 |
| **GHH 偏好**（无财富效应） | $U=\dfrac{\big(C_t-\psi\frac{N_t^{1+\varphi}}{1+\varphi}\big)^{1-\sigma}-1}{1-\sigma}$ | 劳动供给式改为 $\psi N_t^{\varphi}=W_t$（独立于 $C_t$）；常用于开放经济/RBC 拟合 |
| **消费习惯 habit** | 效用含 $C_t-h\,C_{t-1}$ | 欧拉方程出现 $C_{t-1},C_{t+1}$，多一个状态；改善持续性 |
| **投资调整成本** | $K_t=(1-\delta)K_{t-1}+\big[1-S(I_t/I_{t-1})\big]I_t$ | 引入 Tobin's $q_t$ 为独立变量与一条 $q$ 方程；投资对冲击更平滑 |
| **黏性工资**（EHL/Calvo 工资） | 家庭供给差异化劳动并定工资 | 多出一条工资菲利普斯曲线与工资分散度；见 §B 的 Calvo 逻辑（对偶到工资） |
| **两类/异质家庭**（储蓄者+手停口停 HtM、或 HANK） | 各类各一套预算与 FOC，按人口权重加总 | 主体数翻倍；HANK 见 `references/heterogeneity.md` |
| **开放经济** | 预算含外国债券与净出口，加 UIP（非抛补利率平价） | 多 (家庭-外) UIP 式与外国债券状态 |

> 这些**不是你该独自穷举的**：搭家庭前，把「含资本否 / 偏好形式 / 是否异质 / 开放否」做成选择题
> （带推荐）抛给用户（SKILL.md 第1.5步 / 闸门1），定下来再写 derivation.md 的家庭小节。

## B. 厂商：利润最大化问题与一阶条件

**先决一个分叉（A/B，问用户）：一层竞争厂商（RBC 类）还是两层垄断竞争厂商（NK 类）。**
RBC：厂商完全竞争、价格 = 边际成本、无定价权 → 只有要素需求 FOC。
NK：终端厂商（竞争、CES 加总）+ 中间厂商（垄断、黏性定价）→ 多出边际成本与定价方程。

### B.1 RBC：一层完全竞争厂商

厂商每期租资本 $K_{t-1}$、雇劳动 $N_t$，CRS 生产 $Y_t=A_tK_{t-1}^{\alpha}N_t^{1-\alpha}$，逐期静态最大化利润：

$$\max_{\{K_{t-1},\,N_t\}}\; A_t K_{t-1}^{\alpha}N_t^{1-\alpha} - R_t^k K_{t-1} - W_t N_t$$

**一阶条件**（要素边际产品 = 要素价格）：

- **(厂商-1) 劳动需求**：$\;W_t = (1-\alpha)\dfrac{Y_t}{N_t} = (1-\alpha)A_t K_{t-1}^{\alpha}N_t^{-\alpha}$
- **(厂商-2) 资本需求**：$\;R_t^k = \alpha\dfrac{Y_t}{K_{t-1}} = \alpha A_t K_{t-1}^{\alpha-1}N_t^{1-\alpha}$

CRS 下利润为零（$\Pi_t=0$），家庭预算里的 $\Pi_t$ 可略。生产函数 $Y_t=A_tK_{t-1}^{\alpha}N_t^{1-\alpha}$ 作为一条恒等式进 §D。

### B.2 NK：两层厂商（垄断竞争 + Calvo 黏性价格）

**第一层 终端厂商**（完全竞争）把中间品 $Y_t(j)$ 用 CES 加总成最终品，最小化购入成本 → 得：

- **中间品需求**：$\;Y_t(j)=\Big(\dfrac{P_t(j)}{P_t}\Big)^{-\epsilon}Y_t$
- **价格指数**：$\;P_t=\Big(\displaystyle\int_0^1 P_t(j)^{1-\epsilon}\,dj\Big)^{\frac{1}{1-\epsilon}}$

其中 $\epsilon>1$ 为替代弹性，加成 $\frac{\epsilon}{\epsilon-1}$。

**第二层 中间厂商 $j$**（垄断竞争）以 $Y_t(j)=A_t N_t(j)$（或含资本 $A_tK_{t-1}(j)^{\alpha}N_t(j)^{1-\alpha}$）生产。

成本最小化 → **实际边际成本 $MC_t$**（对所有 $j$ 相同）：
- 仅劳动：$\;MC_t = \dfrac{W_t}{A_t}$
- 含资本：$\;MC_t = \dfrac{1}{A_t}\Big(\dfrac{R_t^k}{\alpha}\Big)^{\alpha}\Big(\dfrac{W_t}{1-\alpha}\Big)^{1-\alpha}$，要素比 $\dfrac{R_t^k}{W_t}=\dfrac{\alpha}{1-\alpha}\dfrac{N_t(j)}{K_{t-1}(j)}$

**Calvo 定价**：每期仅 $1-\theta$ 比例厂商可重定价。可调价厂商选 $P_t^{*}$ 最大化未来预期贴现实际利润（受需求约束），FOC 整理为**最优重定价**（标记 $\Pi_t\equiv P_t/P_{t-1}$ 为毛通胀）：

$$\frac{P_t^{*}}{P_t}=\frac{\epsilon}{\epsilon-1}\frac{x_{1,t}}{x_{2,t}},\qquad
\begin{cases}
x_{1,t}=C_t^{-\sigma}\,MC_t\,Y_t + \theta\beta\,E_t\big[\Pi_{t+1}^{\epsilon}\,x_{1,t+1}\big]\\[2pt]
x_{2,t}=C_t^{-\sigma}\,Y_t + \theta\beta\,E_t\big[\Pi_{t+1}^{\epsilon-1}\,x_{2,t+1}\big]
\end{cases}$$

（$x_{1,t},x_{2,t}$ 为两个递归辅助变量，把无穷和写成可计算的递归式——非线性 Calvo 标准写法。）

**价格指数演化**（由 Calvo 抽取规则）：
$$1=\theta\,\Pi_t^{\epsilon-1}+(1-\theta)\Big(\frac{P_t^{*}}{P_t}\Big)^{1-\epsilon}$$

> **FOC 编号**：上述 $MC_t$、$x_{1,t}$、$x_{2,t}$、$P_t^{*}/P_t$、价格指数演化各算一条均衡条件，在
> derivation.md 里继续编号 (F·)。**价格分散度** $\Delta_t$ 进 §D 市场出清（影响总量生产）。
>
> **线性化结果（备查，不直接写进非线性 .mod）**：上述 Calvo 块围绕零通胀稳态对数线性化即得
> **新凯恩斯菲利普斯曲线 NKPC** $\;\pi_t=\beta E_t\pi_{t+1}+\kappa\,\widehat{mc}_t$，其中
> $\kappa=\frac{(1-\theta)(1-\theta\beta)}{\theta}$。仅当复制的论文只给线性系统、或用户要线性版时
> 才直接写这条并 `model(linear)`（R8 例外②）。

### B.3 厂商侧其它常见变体

| 变体 | 怎么改 |
|------|--------|
| **Rotemberg 二次调价成本**（替代 Calvo） | 单一代表性中间厂商，无价格分散度 $\Delta_t$；定价式为含 $\Pi_t(\Pi_t-1)$ 的非线性差分式。非线性二阶福利分析更干净 |
| **黏性工资** | 把 Calvo 逻辑对偶搬到劳动市场，得工资 NKPC 与工资分散度 |
| **可变资本利用率** | 引入利用率 $u_t$ 与成本函数，$R_t^k$ 改写为利用率的函数 |
| **指数化**（价格/工资对上期通胀部分指数化） | Calvo 递归式与价格指数演化里出现 $\Pi_{t-1}$，增持续性 |
| **固定成本 / 进入退出** | 改利润式与零利润条件 |

> 终端是否两层、用 Calvo 还是 Rotemberg、含不含资本——同属 A/B 分叉，先问用户再写。

## C. 政府与央行：政策函数（注意：通常**不是最优化问题**）

**关键区分**：标准（非最优政策）模型里，政府与央行**不解最优化问题**——它们执行**机械的政策规则**。
因此本块产出的是**政策函数 / 预算恒等式 / 外生过程**，**不编 FOC**，在 derivation.md 里归入
第4节（市场出清与恒等式）或第5节（外生过程），而非第2/3节。
（**例外**：若做 Ramsey / OSR / 相机抉择最优政策，则央行*确实*在优化 → 改读
`references/optimal-policy.md`，那里政策由福利最大化的 FOC 决定。）

### C.1 财政（政府预算 + 支出/税收规则）

**政府预算约束**（与家庭预算对应，实际项）：

$$G_t + B_{t-1} = T_t + \frac{B_t}{R_t}$$

闭合方式（**问用户**：李嘉图 vs 含财政规则）：

- **李嘉图 / 一次总付税**（最简，默认）：$B_t=0$ 每期成立，税收残差地满足 $T_t=G_t$。
  此时政府债券退出模型，家庭预算里 $B$ 也消去。
- **含债务的财政规则**：税收对债务反馈以保偿付，如
  $\;T_t-T = \gamma_b\,(B_{t-1}-B)\;$（$\gamma_b$ 足够大保证债务稳定），$B_t$ 成为内生状态。

**政府支出**常设为外生 AR(1)（进第5节 外生过程）：
$$\log G_t = (1-\rho_g)\log G + \rho_g\log G_{t-1} + \varepsilon_t^{g}$$

### C.2 货币政策（央行规则）

**泰勒规则 Taylor rule（非线性/水平形式，含利率平滑）**——$R_t^{n}$ 为**名义**毛利率：

$$\frac{R_t^{n}}{R^{n}}=\left(\frac{R_{t-1}^{n}}{R^{n}}\right)^{\rho_R}
\left[\left(\frac{\Pi_t}{\Pi}\right)^{\phi_{\pi}}\left(\frac{Y_t}{Y^{*}}\right)^{\phi_{y}}\right]^{1-\rho_R}
\exp(\varepsilon_t^{m})$$

其中 $\Pi$ 为通胀目标（稳态毛通胀），$Y^{*}$ 为产出目标（潜在/稳态产出），$\varepsilon_t^{m}$ 为货币政策冲击。

**费雪方程 Fisher equation**（把名义利率接回实际利率，使其与家庭欧拉方程一致）：
$$R_t^{n} = R_t\,E_t\big[\Pi_{t+1}\big]\quad\text{（确定性等价）}$$
家庭欧拉方程里的 $R_t$ 即此实际利率；NK 模型常直接用名义利率写欧拉方程
$\;C_t^{-\sigma}=\beta R_t^{n} E_t\big[C_{t+1}^{-\sigma}/\Pi_{t+1}\big]$，二者等价。

> **BK / 泰勒原则**：$\phi_{\pi}>1$ 才满足 Blanchard–Kahn（决定性均衡）。`check;` 若 BK 不过先查此处。
> **ZLB（名义利率有效下限）**：随机情形**禁用** $\max(\cdot)$ 写 ZLB（R6，摄动在拐点给错导数）→
> 改用 OccBin（`references/occbin.md`）或完全预见 + `lmmcp`。

**常见替代规则（问用户）**：

| 规则 | 写法要点 |
|------|----------|
| 简单泰勒（无平滑） | 去掉 $\rho_R$ 项 |
| 价格水平目标 | 反馈对象换成价格水平偏离 |
| 货币增长规则 | $\Delta\log M_t$ 外生或反馈，配货币需求式（需引入实际余额进效用/CIA） |
| 钉住通胀/汇率 | 开放经济常见，配 UIP |

### C.3 央行/政府侧小结（编号归属）

- 泰勒规则、费雪方程、政府预算、财政规则 → 都是**恒等式/规则**，在 derivation.md 继续编号但归第4节；
- $G_t$（及其它外生政策扰动过程）→ 归第5节 外生过程；
- 这些式子**各定一个内生变量**（如泰勒规则定 $R_t^n$、费雪式定 $R_t$、政府预算定 $T_t$ 或 $B_t$），
  仍要计入 R4 的「方程数 = 变量数」核对——**不优化≠不算方程**。

## D. 市场出清与总量恒等式

把各 block 加总、令供需相等，闭合模型。这些**不是 FOC**（无最优化），但仍编号、仍计入 R4。

### D.1 各市场出清条件

- **产品市场（资源约束）**——封闭经济：
$$Y_t = C_t + I_t + G_t$$
（无资本时 $I_t=0$；开放经济加净出口 $NX_t$。）

- **劳动市场出清**：$\;N_t=\displaystyle\int_0^1 N_t(j)\,dj$（RBC 单一厂商时退化为 $N_t^{需}=N_t^{供}$，常已隐含在要素需求里，可不单列）。

- **资本市场出清**：$\;K_{t-1}=\displaystyle\int_0^1 K_{t-1}(j)\,dj$（同上，单一厂商时常隐含）。

- **债券/资产市场出清**：代表性家庭封闭经济 $\;B_t=0$；含政府债务则家庭持有 = 政府发行。

### D.2 生产函数与（NK）价格分散度

- 生产函数本身是恒等式，进 §D：$\;Y_t=A_tK_{t-1}^{\alpha}N_t^{1-\alpha}$（RBC）。
- **NK 价格分散度 $\Delta_t$**：黏性价格下各厂商产量不同，总投入与总产出间有楔子：
$$Y_t\,\Delta_t = A_t K_{t-1}^{\alpha}N_t^{1-\alpha},\qquad
\Delta_t=(1-\theta)\Big(\frac{P_t^{*}}{P_t}\Big)^{-\epsilon}+\theta\,\Pi_t^{\epsilon}\,\Delta_{t-1}$$
$\Delta_t\ge 1$，稳态零通胀时 $\Delta=1$。$\Delta_t$ 是状态变量、各算一条方程。

### D.3 Walras 定律（**R4 关键：必须省掉一条**）

所有市场出清 + 所有预算约束**线性相关**：任一市场的出清可由其余市场出清 + 各预算约束推出。
因此**必须从模型中删去一条冗余的市场出清式**（通常删债券/资产市场或某一要素市场），否则
方程数 > 变量数、系统过定，Dynare 报错或奇异。

> **实操**：通常**保留资源约束 $Y_t=C_t+I_t+G_t$，删掉债券市场出清**（$B_t=0$ 已由家庭/政府预算隐含）。
> 写完模型块**数一遍方程 = 内生变量**（R4）；多一条先查是不是 Walras 冗余。

### D.4 外生过程（归第5节，但建模时一并想好）

每个结构冲击对应一条 AR(1)（**varexo 只放创新项**，R3）：
$$\log A_t=\rho_a\log A_{t-1}+\varepsilon_t^{a},\quad
\log G_t=(1-\rho_g)\log G+\rho_g\log G_{t-1}+\varepsilon_t^{g},\quad
v_t=\rho_v v_{t-1}+\varepsilon_t^{m},\;\dots$$
持续过程（$A_t,G_t,v_t$）是**内生变量**，只有 $\varepsilon_t^{\cdot}$ 进 `varexo`。
