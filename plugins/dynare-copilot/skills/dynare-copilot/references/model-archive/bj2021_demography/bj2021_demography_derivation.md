## §1 模型概述

三部门 + 家庭的内生增长 OLG：
- **最终品**：CES 聚合连续统中间品 `i∈[0,Z_t]`，替代弹性 ε（>1）。
- **中间品两部门**：
  - 劳动密集 `i∈Z_t\A_t`：`y = (K^α L^{1-α})^{1-γ_I} τ^{γ_I}`
  - 自动化 `i∈A_t`：`y = (K^α (ζ M)^{1-α})^{1-γ_I} τ^{γ_I}`，机器人 M 替代劳动，ζ=机器人相对生产率。
  能自动化的产品一定用机器人（机器人比劳动更高产）。
- **R&D**：创新（造新产品 Z↑，新品只能先用劳动产）+ 自动化（把已有品转成可用机器人产，A↑）。两者
  争夺有限的 R&D 劳动与贷款 → **创新/自动化权衡**（论文核心）。
- **机器人生产**：投入 Φ 单位最终品产出 `M = ε̄ Φ^χ`，**χ<1** 是防止"无劳动奇点"的瓶颈约束。
- **家庭**：Gertler(1999) 两阶段（工人 w / 退休 r），Epstein-Zin 递归效用（风险中性 + 正跨期替代）。
  完全年金市场。人口动态由生育 ψ_y、退休 1-ω、退休者存活 ψ_r 驱动。

**增长来源**：新品种 Z↑（生产率+再配置就业）与自动化 A↑（生产率+替代就业）。
BGP：`g = g_n`（总产出按人口增速长？见 §3 命题1 实际是 `g^{1-χ}=g_n`），人均增长 = `g_n^{χ/(1-χ)}`。

## §2 各主体最优化问题（详见 paper.txt §2 与附录 A）

- **最终品商**：`max` 利润 → 需求 `y_i = (p_i/p)^{-ε} y`。
- **中间品商**（两部门）：给定产量最小化成本 `T C = p q M + (r^k+δ)K + P τ`（自动化）/
  `= W L + (r^k+δ)K + P τ`（劳动）。→ 要素需求 (A.1)–(A.4)、(A.8)–(A.11)，相对价 (A.5)/(A.12)。
- **创新者**：投 S、雇 L_I 造新品，得新品利润份额 ν。FOC (11)+(A.20)/(A.21)。
- **自动化者**：投 φ、雇 L_A 把劳动品转自动化品，成功概率 σ_t，得自动化品利润份额 ν。FOC (A.23)–(A.25)。
- **机器人商**：`max q M − Φ`，s.t. `M=ε̄Φ^χ` → (A.15)–(A.17)，`q = Φ/M·(1/χ)`（曲率定价）。
- **家庭**：工人/退休 Epstein-Zin 递归，预算约束 (3)。→ MPC 工人 Φ_t、退休 Π_t（>Φ），
  消费函数 (6)/(A.42)/(A.33)，MPC 递归 (A.44)/(A.35)，人力财富 (A.45)、红利现值 (A.46)。

## §3 关键 FOC / BGP 关系（命题1，χ<1，有技术进步）

记 `g=Y_t/Y_{t-1}`，`g_n=N_t/N_{t-1}`，`g_q=q_t/q_{t-1}`，`g_wg=W_t/W_{t-1}`。

- **F1（相对价同增）** 两部门相对价在 BGP 同速增 → `g_q = g_wg`（A.69d/A.69l + 16/17）。
- **F2（劳动供给）** 每人 1 单位有效劳动 → `g_L = g_n`，于是 `g_q = g_wg = g/g_n`。
- **F3（机器人生产 BGP）** `M=ε̄Φ^χ` + 机器人份额常数 → `g^{-χ} g_q = 1`。
- **F1–F3 合并** → `g^{1-χ} = g_n` ⇒ **人均增长 `g/g_n = g_n^{χ/(1-χ)}`**（推论2 的引擎）。
- **F4（品种增长）** `g_Z = g_A = g^{(1-α)(ε-1)(1-ρ̂)(1-γ_I)}`（ρ̂ 见 §7 歧义）。
- **A1**：`a_z ≤ ā`（自动化品占比上界，校准下 ≤75%）。
- **A2**：`ε > 1 + 1/[(1-α)(1-ρ̂)(1-γ_I)]`（markup<23%）。

## §4 市场出清与 Walras 定律检查

出清（A.60–A.66 / A.72f–l）：最终品 `1 = c+i+s+φ+τ̃+...`、资本 `k=k_m+k_L`、
投入 `τ̃=τ_m+τ_L`、机器人 `M=A·m_j`、劳动 `N^{wRD}=L_I+L_A`、`N^{wL}=L`、品种份额 `1=y_m+y_L`。

**Walras 冗余**：金融中介零利润 (A.55, `F_t=0`) + 工人/退休资产流 (A.58/A.59) + 各主体预算约束
联立后，**最终品资源约束 (A.72g) 冗余**（由其余条件线性组合得出）。
→ **资源约束不进 model 块**（或进、则剔除资产流之一）。实现时二选一，避免 BK 因方程线性相关失败。

## §5 外生过程（确定性转移的驱动）

无随机冲击。外生 = **人口参数**，在 initval（旧 BGP）与 endval（新 BGP）间做永久跳变：
- `psi_y`（生育率 n）：US 0.0265→0.0236；EU 0.0253→0.0206
- `psi_r`（退休者存活 = 1−死亡率）：US 0.93→0.963；EU 0.94→0.976
- `omega` 固定 = 1−1/45 = 0.97778（退休概率 1/45）。

实现按 `Solow_growth_rate_changes.mod` 模式：把 psi_y、psi_r 设为 `varexo`，
`initval` 给 1993 值并解旧 BGP，`endval` 给 2055 值并解新 BGP，`perfect_foresight_solver` 求转移。

## §6 稳态（BGP）求解顺序

BGP 上去趋势变量为常数。求解链（给定人口 → g_n）：
1. 人口块：`g_n=g^w=ω+ψ_y`，`τ^r=(1-ω)/(g^w-ψ_r)`，份额。
2. 增长：`g=g_n^{1/(1-χ)}`，`g_q=g_wg=g/g_n`，`g_Z=g_A`（F4），`a_z`（由 A.70b 定常：`g_A=σ(1/a_z−1)+φ`）。
3. 利率 R：由家庭 MPC 稳态 (A.67g/h) 与资产市场联立（数值解一维）。
4. 要素价：`r^k = R−1+δ`（A.71a）；`W`、`q` 由相对价与份额（A.69i lst）定。
5. 部门份额 `y_m,y_L`（1=y_m+y_L），R&D 配置 `s,φ,l_i,l_a`，值 `v,j`，消费份额。
*多数为解析/半解析；R 与份额可能需 1–2 维数值根（steady_state_model 内 back-solve，参照 trend_rbc_gov_inv）。*
